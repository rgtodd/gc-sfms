Unless otherwise noted, all REST services support the following operations:

URL Operation Request Response Description
/ GET N/A SearchResult<TEntity> Retrieves all entities.
/TKey GET N/A TEntity Retrieves entity associated with the specified key.
/ PUT TEntity CreateResult<TKey> Creates a new entity.
/TKey PUT TEntity UpdateResult<TKey> Updates an entity.
/TKey DELETE N/A DeleteResult<TKey> Deletes an entity.
